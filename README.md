# Cardiovascular Performance Analysis Using ANOVA

A statistical analysis project investigating how demographic and clinical variables influence maximum heart rate using one-way and multi-factor ANOVA.

## Executive Summary
This project evaluates the relationship between cardiovascular performance and key predictors such as sex, chest pain type, and other clinical indicators. Using structured health data, the analysis applies ANOVA-based statistical modeling, validates model assumptions, and identifies significant factors associated with maximum heart rate.

## Problem Statement
Understanding which patient characteristics are significantly associated with cardiovascular performance can improve clinical interpretation and support evidence-based analysis of heart health indicators.

## Objectives
- Quantify the effect of selected demographic and clinical variables on maximum heart rate
- Test whether observed group differences are statistically significant
- Validate ANOVA assumptions to ensure reliability of inference
- Produce interpretable findings from structured clinical data

## Methodology
The analysis includes:
- One-way ANOVA
- Multi-factor ANOVA
- Assumption testing
- Variance homogeneity checks
- Significance testing at p < 0.05

## Tools
- R
- Statistical analysis
- ANOVA
- Data visualization

## Key Outcome
This study identifies statistically meaningful relationships between selected predictors and maximum heart rate, demonstrating the use of ANOVA for interpretable analysis of clinical data.
